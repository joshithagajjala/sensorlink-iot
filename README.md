# SensorLink

### Real-Time IoT Monitoring & Alerting Platform

SensorLink is an IoT monitoring platform designed to track sensor data from ESP32 devices in real time using MQTT communication.

The system enables users to register devices, configure alert thresholds, monitor sensor readings, and manage multiple devices through a centralized dashboard.

## Features

* Real-time sensor monitoring
* Device registration and management
* Threshold-based alert system
* Multi-device support
* Device status tracking
* Responsive web dashboard
* MQTT-based communication
* ESP32 integration

## Technology Stack

### Hardware

* ESP32
* HC-SR04 Ultrasonic Sensor

### Software

* Next.js
* React
* TypeScript
* Tailwind CSS
* MQTT

## Project Architecture

ESP32 Sensor → MQTT Broker → SensorLink Dashboard
